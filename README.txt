Our system only runs on linux

You need to have python3 installed

You need to install Flask
this can be done by going to the root and typing:
sudo pip3 install flask

To run the sytem navigate to the folder and type the following command:
python3 app.py
then ctrl+click on the http://0.0.0.0:5000/ link

To stop the program press ctrl+C in the command line
